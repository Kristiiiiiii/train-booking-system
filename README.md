# 🚆 Train Booking System

A simple frontend project that simulates a train ticket booking system. This project allows users to enter details and book tickets through a clean and user-friendly interface.



## 📌 Features

* User-friendly booking form
* Input fields for passenger details
* Simple and clean UI design
* Responsive layout (if applicable)



## 🛠️ Tech Used

* HTML
* CSS (if used)

## 📂 Project Structure

```
train-booking-system/
   index.html
```

## 🚀 How to Run

1. Download or clone the repository
2. Open the folder
3. Double-click on `index.html`
4. The project will open in your browser


## 📖 Description

This project is designed as a basic simulation of a train booking interface. It helps in understanding form handling, layout design, and frontend development concepts.
